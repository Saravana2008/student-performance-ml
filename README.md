# student-performance-ml
A Machine Learning project to predict student academic performance using Logistic and linear Regression and data analysis techniques.

🎓 Student Performance Prediction

This project uses Machine Learning to predict student final grades based on various factors such as study habits, attendance, and academic scores.

🚀 Features
- Data preprocessing and cleaning
- Handling categorical variables using one-hot encoding
- Logistic Regression model for classification
- Performance evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - Confusion Matrix

 📊 Dataset Features
- Study hours
- Attendance percentage
- Subject scores (Math, Science, English)
- Internet access
- Study methods
- Parental education

🧠 Model Used
- Logistic Regression
- Linear Regression 
## 📈 Results
- Accuracy: 75.56%
- Precision: 0.7569
- Recall: 0.7556
- F1 Score: 0.7560

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib (optional)

## 🎯 Goal
To understand how different factors affect student performance and build a predictive model.
